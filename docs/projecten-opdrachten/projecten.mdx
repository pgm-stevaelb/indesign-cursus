# Eindwerk project

:::caution Indienen
Indienen van je project doe je telkens via Teams. Gelieve ook telkens de einddatum te respecteren.
:::

Als eindwerk van het project, maken we een **magazine met onderwerp naar keuze** van minimul 8 pagina's. Alle verworven kennis zal voor deze opdracht nodig zijn.

:::danger Leerkrachten veto
Volgende onderwerpen mogen **NIET** gebruikt worden voor jouw magazine.
- Voetbal (tenzij je een zeer interessant magazine kan maken over )
:::

## Overizht opdracht

### Start hier

- [ ] Kies een **onderwerp** (je hobby, je passie, probeer eens iets totaal anders en creatiefs)
- [ ] Start je document en maak volgende alvast in orde voor je begint:
  - [ ] Gebruik een **duidelijk formaat** (je mag afstappen van A4, maar zorg dat het geen té klein of té groot document wordt)
  - [ ] Zorg voor **voldoende afloop** en dat je afloop **verschilt aan de binnenkant** van je document (denk aan de plooi van je magazine)
  - [ ] Werk met **kolommen en hulplijnen** (denk aan de grid-layout)

### Doe dan dit

Zorg ervoor dat volgende elementen in je project aanwezig zijn:
- [ ] **<ins>Minstens</ins> 2 verschillende stramienen**
  - [ ] Stramien voor voor- en achterpagina (cover -> zonder paginanummering)
  - [ ] Stramien voor alle andere pagina's (met automatische paginanummering)
- [ ] **Tekst en afbeeldingen**. Vermeld telkens bron <ins>buiten de witruimte</ins> indien niet eigen tekst en afbeeldingen.
- [ ] **Kleurenpalet**. Stel een eigen kleurenpalet op en gebruik die in je document.
- [ ] **Google Font**. Installeer minstens één Google Font. Zorg ervoor dat wij als leerkracht dit lettertype <ins>niet hoeven te downloaden</ins>.

:::tip
- Gebruik verschillende lettertypes / fonts voor je titels en doorlopende teksten.
- Zorg voor allemaal **leesbare fonts**
:::

- [ ] **Stijlen: <ins>Minimum</ins> het volgende:**
  - [ ] **Alineastijlen**: Voor elk type alinea moet er een stijl zijn (titels, subtitels, doorlopende tekst, ...) 
  - [ ] **Tekenstijlen**: <ins>Minimum 2</ins> types tekenstijlen (vette tekst, onderlijnde tekst, gekleurde tekst, ...)
  - [ ] **Objectstijlen**: Zorg voor <ins>minstens 1</ins> objectstijl. (schaduw op een kader, transparatie van een vlak, ...) 
- [ ] **Tekstomloop**: Maak op een correcte wijze gebruik van tekstomloop (denk aan afstand tussen tekst en afbeeldingen), en zorg ervoor dat <ins>minstens 1 afbeelding een omloop krijgt op basis van zijn alfakanaal</ins>. 

## Indienen

- [ ] Exporteer het InDesign bestand als **.pdf met snijtekens en afloopinstellingen**.
- [ ] Maak een **pakket** van je InDesign file. 
- [ ] Plaats alles in een gezipte map met <ins>**jouw naam en klas**</ins> op Teams **voor de deadline**.

:::danger Voorstelling
Je werk zal getoond worden aan de rest van de klas op het einde van de cursus. Zorg dus voor iets dat iedereen kan interesseren en dat je uitblinkt!
:::
