# Opdracht 1 - Recept

:::tip Wat leren we met deze opdracht?
- Een bestaand bestand openen
- Tekst aanpassen
- Afbeeldingen aanpassen
- Kleuren aanpassen
:::

## Introductie

Het eerste project is een inleidend project. We passen een bestaand recept aan.

## Start

- Download de startdocumenten [hier](@site/static/opdrachten/opdracht-1.zip)
- Pak alles uit en sla op in een map op je **OneDrive**

```
📦 opdracht-1.zip
┣━ 📋 start.indd
┣━ 📄 teksten.docx
┗━📂 assets
  ┣━ 🖼️ appeltaart.jpg
  ┣━ 🖼️ tiramisu.jpg
  ┣━ 🎭 taart-icoon.svg
  ┗━ 🎭 gebak-icoon.svg
```

## De opdracht

### Foutcontrole

:::tip Foutcontrole
Maak zoveel mogelijk gebruik van de preflighting functie. Deze vertelt je letterlijk wat er mis gaat. Bekijk het hoofdstuk [4.6 Preflighten](#) voor meer informatie.
:::

### Aanpassen van teksten en afbeeldingen

Start de opdracht door het openen van **start.indd**

Je ziet een recept voor het maken van <ins>tiramisu</ins>.

Pas de elementen en teksten aan zodat het een recept wordt voor het maken van een <ins>appeltaart</ins>. Het recept kan je vinden in `teksten.docx`, de afbeeldingen kan je vinden onder `assets`.

:::danger Escaleren
Wanneer je meer uitdaging wenst, of gewoon een uitbreiding op de opdracht kan je volgende stappen proberen:
- Zoek een eigen recept. Verwijder alle bestaande inhoud en plaats je eigen recept in de plaats. Pas de afbeeldingen en de tekst aan (Let op: bij afbeeldingen: zorg dat ze groot genoeg zijn en [omgezet](../extra/9-2-omzetten) zijn voor correct gebruik)
:::

Vergeet niet je bestand op te slaan.
